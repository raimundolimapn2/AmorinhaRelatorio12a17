Resumo Executivo

No período analisado, a campanha de Meta Ads entregou desempenho consistente com 132.420 impressões e 69.900 pessoas alcançadas, sustentados por investimento de R$ 814,45. A audiência está bem segmentada (82% mulheres, faixa 25-34 anos predominante) e concentrada em mobile. Identificou-se oportunidade crítica: uma campanha de tráfego opera com CTR 76% abaixo da média da conta. Realocando verba para criativos com melhor tração e otimizando para o horário de pico (15h-21h), projeta-se aumento de 25-35% na eficiência.

Insights Principais

🎯 Desempenho Desigual

Campanha '[Trafego] - Mineiros - [Loja] 29/05' opera com CTR 76% abaixo da média (0,11% vs 0,98%). Apesar do alto volume de impressões, baixo CTR indica criativos inadequados.

📱 Mobile é Dominante

100% do alcance vem de aplicativos móveis. Todos os ajustes devem otimizar para mobile: primeiros 2 segundos críticos, verticais preferidas, copy direto.

👩 Segmentação Forte

Base fortemente concentrada em mulheres 25-34 anos (62,8% do público). Segmento bem definido que responde bem a campanhas. Oportunidade de criar variações específicas.

⏰ Janela de Ouro

Entre 15h-21h concentram-se 42% das impressões totais. Testes de criativo devem priorizar este horário para melhor aproveitamento do leilão de anúncios.

🚀 Melhor Performer

Campanha '[Trafego] - Mineiros' com 3,96% CTR e R$ 0,12 CPC é a estrela. Combina volume (147 cliques) com excelente eficiência. Modelo para escalas futuras.

💬 Instagram Dominante

82 das 84 conversas iniciadas vieram do Instagram. Facebook está subperformando. Realocar verba prioritariamente para Instagram nas próximas campanhas.

Recomendações Estratégicas

🔴 Alta Prioridade 1. Realocar Verba para Criativos com Melhor Tração
✓ Ação: Reduzir investimento na campanha crítica (CTR 0,11%) e realocar para 'Post Instagram: A AMORINHA reinaugurou...' que combina R$ 0,67 CPC com 434 cliques

A concentração de verba em criativos com baixo CTR está puxando a eficiência geral da conta para baixo. O padrão indica que o problema não é orçamento, mas alocação.

📈 Impacto esperado: +25-35% em eficiência geral de cliques
🔴 Alta Prioridade 2. Otimizar Todos Criativos para Mobile
✓ Ação: Produzir criativos em formato vertical (9:16), copy visível nos primeiros 2 segundos, CTA claro. Preparar 2-3 variações de copy para testes simultâneos

Com 100% do alcance vindo de apps, cada aspecto do criativo deve estar otimizado para essa realidade. Teste A/B de copy nos próximos 7 dias.

📈 Impacto esperado: +15-20% em CTR
🟠 Média Prioridade 3. Segmentar Público por Perfil Específico
✓ Ação: Criar conjuntos de anúncios separados para Mulheres 25-34 (foco inovação) e Mulheres 18-24 (foco lifestyle), antes de aumentar frequência

A frequência média está em 1,89, indica espaço para escalas. Mas primeiro, criar mensagens específicas para cada segmento etário do público principal.

📈 Impacto esperado: +10-15% em conversão
🟠 Média Prioridade 4. Concentrar Testes no Horário de Pico (15h-21h)
✓ Ação: Executar 70% do orçamento de testes criativos entre 15h-21h. Usar day parting para escalas maiores apenas neste window

O horário de pico não é apenas onde há mais volume, é onde o leilão de anúncios tem dinâmica diferente. Dados de teste melhoram quando rodados ali.

📈 Impacto esperado: +10-15% em ROI de testes
🟢 Baixa Prioridade 5. Expandir para Novos Públicos (Pós-Otimização)
✓ Ação: Após otimizar criativos e público principal, testar públicos masculinos com ângulos diferentes (grupos de amigos, celebração) e novas regiões

Expansion horizontal deve vir após vertical está otimizada. Primeira semana: corrigir criativos. Segunda semana: expandir públicos. Terceira: novos mercados.

📈 Impacto esperado: +20-30% em volume total de leads
